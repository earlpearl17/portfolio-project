For this tutorial, I am using a Windows 7 Professional Service Pack 1 host machine. I will be using the Etcher utility to create a bootable USB for CentOS 7. The bootable USB can be used to install the CentOS 7 operating system on a physical PC or in a VirtualBox VM (virtual machine).

After you've completed this tutorial, you can access one of my other tutorials where I demonstrate the installation of CentOS 7 using a bootable USB in a VirtualBox VM. It can be accessed here.


Steps to complete tutorial:

  1. Download Etcher
  2. Install Etcher
  3. Create a bootable USB or DVD
Download Etcher

Go to the Etcher site at https://www.balena.io/etcher/ and download the application.



Install Etcher

Once downloaded, double-click on the executable to begin installation.



Accept the security warning and click Run.



Accept the license agreement and click I Agree.



Etcher is installed and ready to be used.



Create a bootable USB or DVD

Let’s perform a test to confirm that it’s working properly. We’ll create a bootable USB from a CentOS 7 ISO.
(Note that if you would rather a DVD, make the necessary changes when selecting a target).

First, on the application console, click the Flash from file button and navigate to the ISO you wish to create a bootable USB from, and click Open



First, on the application console, click the Flash from file button and navigate to the ISO you wish to create a bootable USB from.



Once you've selected your ISO image, click Open



Next, insert your empty USB into an open port (or blank DVD into your optical drive) and click the Select target button to choose the location you wish to save the flashed image.



Ensure the correct destination target is selected (your empty USB or blank DVD), and click the Select (1) button.



Click the Flash! button to create the bootable USB stick.



Etcher is in the process of creating your bootable USB.



Once the operation has completed, you can use your bootable USB to install the CentOS 7 operating system on a physical PC or in a VirtualBox VM (virtual machine).
